Angular presentation

Angular is a nodeJS framework that helps you developing SPA (Single Page Application). It’s one of the three most used framework in 2018, with React and VueJS.
In this tutorial I will explain:

  • What is a SPA

  • Introduction to Angular Components

  • How to create your first Angular App

Explanation of what a SPA is :


A Single Page Application is a web application using a single page to navigate throughout the application. A SPA avoids the reloading of the browser page for each action of the user, would it be wholly or partially.
Angular uses the Model-View-Controller architectural pattern.




source: https://en.wikipedia.org/wiki/Model%E2%80%93view%E2%80%93controller

The View is the structure of the graphic interface that is shown to the user. It’s updated when the Model is modified. The Model is the module that contains the data and logic of your application. A modification of the Model's data results in an update of the View. The Controller is the module that treats what the user does and modifies the Model’s data consequently.

Angular Components:


A component in angular is an object that contains his own part of the application code. It allows you to divide your application in several “blocks” that can be linked with each others.



source: https://docs.angularjs.org/guide/component-router

Each component has its own data, distributed in 4 files:



  • components.css file that contains the own css of the component

  • component.html file that contains the own HTML of the component

  • component.spec.ts file are for unit used for the test of components

  • component.ts file that allows you to use variables and export some class specific to the component

How to create your first Angular App :



  1. Install Node
  2. Install Angular
  3. Create your first application

Angular is a framework of NodeJs, so you need to Install NodeJs in order to use Angular.
Go to this Link https://nodejs.org/en and download NodeJs.
Install It (you may have your own preferences, so I let you manage the installation process as you desire). Now open the Node.js.Command prompt that has been installed. You can find it right in your search Bar.

From now on you need to Install the latest version of Npm. Npm is a package manager for javascript developers. It’s the world largest software registry. We will need it in order to install and update Angular’s features. Use the command :

- npm install npm@latest -g


You can go to Npm official website if you’re willing to discover more about it : https://www.npmjs.com
Install the angular Cli with

- npm install -g @angular/cli


Now you have everything needed to create your first app.
Go to the folder you want to create your angular project in with the cd command: cd [the path to your folder]. and follow these 3 steps:

  1. - Create the application with the command

    ng new [your app name]

    This command entirely creates an application

    2. It may take some time



  2. - Go to the folder of the application you just created with the command :

    cd [your app name]

  3. - Launch your app with ng serve --open (this command launch a local web server that shows your application in your default browser)

You can now go check your app on the default url http://localhost:4200 :
